👉 中文 | English

这是 Hugo 主题 FixIt 的快速启动模板。它使用 Hugo 模块 功能加载主题。

它提供了基本的主题结构和配置。已经设置了 GitHub Actions，可以自动部署博客到 GitHub Pages。此外，还有一个定时任务，每天自动更新主题。
