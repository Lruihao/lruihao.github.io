```xml
<!--[if lt IE 9]>
加载 CSS1
<!--[else]>
加载 CSS2
<![endif]-->
```

这样有效是有效，但是用 HTML VALIDATOR 里，报错，因为这个不符合 XHTML 1.1 的规范，
如果把 ELSE 语句去掉，则正确。

```xml
加载 CSS2
<!--[if lt IE 9]>
加载 CSS1（可以把要重写的写在这里）.
<![endif]-->
```
