宅音乐播放器，HTML5 网页播放器，集成后台管理及 API 调用，目前正在开发中，敬请关注~
原项目 由 IT 技术宅 开源，使用 thinkPHP 开发后台。 fork 地址 是我个人学习模仿的库，也是相当于备份源码。
注：插件修改于明月浩空免费版，仅用于学习交流，无商业价值，如发现商业传播，将禁止软件的免费使用。

1 技术栈

  • 前端：layui
  • 数据库：mysql
